Frequently Asked Questions

Does this module work with custom models?
Yes, the module works with all Odoo models, including custom models created by other modules. It automatically discovers and allows you to control access to any model in your Odoo instance.
Is technical knowledge required?
No, the module is designed for business users. All configuration is done through an intuitive interface. No coding or technical expertise required.
What support is included?
We provide 60 days of free support for any bugs or issues. Additional support is available via email, WhatsApp, and phone. Our support team typically responds within 24 hours during business days.
How does this differ from Odoo's built-in security?
Unlike Odoo's complex technical security groups, this module uses business-focused user groups. You can create groups like "Sales Team" or "HR Staff" and manage permissions through an intuitive interface without dealing with technical security concepts.
Can I control field-level access?
Yes! You can make fields invisible, read-only, or required for specific user groups. You can also control access to relational fields and their options, providing granular control over data visibility.
Does it work with multi-company setups?
Absolutely! The module supports multi-company environments. You can apply access rules to specific companies or across all companies, giving you complete control over user access in complex organizational structures.
Can I hide buttons and tabs dynamically?
Yes! The module automatically discovers buttons, tabs, and other UI elements from your views and allows you to hide them for specific user groups. This creates cleaner, more focused interfaces for different roles.
What happens to existing permissions?
The module works alongside Odoo's existing security system. It adds an additional layer of control without interfering with your current permissions. You can gradually migrate to the new system or use both simultaneously.
Is there a performance impact?
The module is optimized for performance and has minimal impact on your Odoo instance. Access rules are cached efficiently, and the system scales well from small teams to large enterprises with thousands of users.
Can I export/import access rules?
Yes! You can export your access control configurations and import them into other Odoo instances. This makes it easy to replicate access setups across different environments or share configurations with other organizations.
